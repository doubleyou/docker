====================================
``stop`` -- Stop a running container
====================================

::

   Usage: docker stop [-h] [-t value] CONTAINER [CONTAINER...]

   Stop a running container

    -h, --help        Display this help
    -t, --time=value  Number of seconds to wait for the container to stop before
		      killing it.
