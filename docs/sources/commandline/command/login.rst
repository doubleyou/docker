============================================================
``login`` -- Register or Login to the docker registry server
============================================================

::

   Usage: docker login [-h] [-e email] [-p password] [-u username]

   Register or Login to the docker registry server

    -e, --email=email
    -h, --help         Display this help
    -p, --password=password
    -u, --username=username
